## Что такое ratatui-markdown?

ratatui-markdown — это многофункциональная библиотека рендеринга для терминальных пользовательских интерфейсов, построенная на [ratatui](https://github.com/ratatui/ratatui). Она предоставляет четыре основных функциональных модуля, которые можно использовать независимо или комбинировать через виджет `MarkdownPreview`.

## Основные Модули

### Рендеринг Markdown

Парсинг и рендеринг Markdown в стилизованный терминальный вывод:

- **Заголовки**: H1 (`#`), H2 (`##`), H3 (`###`)
- **Абзацы** с автоматическим переносом строк с учётом CJK
- **Встроенное форматирование**: `**жирный**`, `*курсив*`, `***жирный+курсив***`, `` `встроенный код` ``
- **Блоки кода** с опциональными метками языка (блоки mermaid пропускаются)
- **Цитаты** (`>`)
- **Неупорядоченные списки** (`-`, `*`, `+`) и упорядоченные списки (`1.`, `2.`)
- **Горизонтальные линии** (`---`, `***`, `___`)
- **Таблицы** с пропорциональной шириной столбцов и переносом ячеек

### Вид Сворачиваемого Дерева

Парсинг и интерактивная навигация по структурированным данным:

- Парсинг **JSON** и **TOML** в сворачиваемые деревья
- **Развернуть / Свернуть** отдельные узлы, развернуть все, свернуть все, развернуть по глубине
- **Стилизованные ключи**: режим JSON (ключи в кавычках + `:`) или режим TOML (ключи без кавычек + `=`)
- **Клавиатурная навигация**: выбор и переключение на основе курсора
- **Цвет по типу значения**: строки, числа, булевы значения, null — каждый со своим цветом темы

### Гибридная Система Прокрутки

Умная прокрутка, обрабатывающая как свободное перемещение, так и навигацию по элементам:

- **Режим свободной прокрутки**: свободная прокрутка содержимого
- **Режим вовлечения**: автоматически активируется при входе фокусируемых элементов в область просмотра
- **Курсорная навигация**: перемещение между фокусируемыми элементами с клавиатуры
- **Индикатор курсора**: визуальный префикс `> ` на вовлечённых строках
- **Полоса прокрутки**: наложение на основе стрелок
- **Постраничная прокрутка**: поддержка `page_up` / `page_down`

### Виджет MarkdownPreview

Высокоуровневый виджет, объединяющий всё вместе:

- Рендеринг Markdown-контента, деревьев и элементов действий в едином прокручиваемом макете
- **Кэширование**: перестраивает вывод только при изменении содержимого, ширины или поколения темы
- **Удаление преамбулы TOML**: автоматически удаляет преамбулу TOML, ограниченную `+++`
- **Элементы действий**: элементы с метками, выбираемые с клавиатуры, с ID действий
- Делегирует всю навигацию `HybridScrollView`

## Быстрый Старт

```toml
[dependencies]
ratatui-markdown = "0.1"
```

```rust
use ratatui_markdown::preview::MarkdownPreview;

let mut preview = MarkdownPreview::new();
preview.set_content("# Привет, мир!\n\nЭто абзац.");
// рендеринг и обработка ввода в цикле приложения ratatui
```

## Функциональные Флаги

Все функции включены по умолчанию. Отключите функции по умолчанию, чтобы включить только необходимые:

```toml
[dependencies]
ratatui-markdown = { version = "0.1", default-features = false, features = ["markdown"] }
```

| Функция    | Зависимости        | Описание                                          |
|------------|--------------------|---------------------------------------------------|
| `markdown` | —                  | Парсер и рендерер Markdown                        |
| `scroll`   | —                  | HybridScrollView, прокручиваемые списки, полоса прокрутки |
| `tree`     | `scroll`, `serde_json`, `toml` | Сворачиваемое дерево JSON/TOML         |
| `preview`  | `markdown`, `scroll`, `tree` | Унифицированный виджет MarkdownPreview |

## Структура Проекта

```
ratatui-markdown/
  src/
   ├── lib.rs                  # Точка входа: модули с функциональными шлюзами
   ├── theme.rs                # Трейт RichTextTheme, токен Generation
   ├── constants/
   │   ├── mod.rs              # Реэкспорты
   │   ├── box_chars.rs        # Константы символов рамок
   │   └── list_prefix.rs      # Соединители деревьев, стрелки, маркеры
   ├── markdown/
   │   ├── mod.rs              # Структура MarkdownRenderer
   │   ├── parser.rs           # Блочный парсер Markdown
   │   ├── types.rs            # Перечисление MarkdownBlock, TextToken
   │   ├── render.rs           # Блочный рендерер (+ таблицы)
   │   ├── inline.rs           # Парсер встроенного форматирования
   │   └── text.rs             # Перенос текста с учётом CJK
   ├── scroll/
   │   ├── mod.rs              # Реэкспорты
   │   ├── hybrid_scroll/      # HybridScrollView (основной виджет)
   │   ├── scrollable_list.rs  # Обобщённый ScrollableList<T>
   │   ├── scrollable_panel.rs # Простой помощник прокрутки
   │   ├── focusable_list.rs   # Рендерер FocusableItemList
   │   ├── follow_scroll.rs    # FollowScrollState
   │   └── scrollbar.rs        # Виджет ArrowScrollbar
   ├── tree/
   │   ├── mod.rs              # Реэкспорты
   │   ├── tree_lines.rs       # Построение линий дерева
   │   └── collapsible_tree/   # CollapsibleTree + операции + рендеринг
   └── preview/
       └── mod.rs              # Унифицированный виджет MarkdownPreview
```
